= Configuring Networking in OpenShift
:page-layout: article

[WARNING]
====
Incorrect network configuration can disrupt cluster communication.
====

== Network Policies

[source,yaml]
----
kind: NetworkPolicy
apiVersion: networking.k8s.io/v1
metadata:
  name: deny-all
spec:
  podSelector: {}
  policyTypes:
  - Ingress
----
